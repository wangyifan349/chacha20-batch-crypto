# chacha20-batch-crypto ⚡️🔒
基于 [PyCryptodome](https://www.pycryptodome.org/) 的高性能文件夹批量加密/解密工具，支持多线程加速，安全、简单、易用，专为本地多文件敏感数据保护设计。

作者: [wangyifan349](https://github.com/wangyifan349)

---

## ✨ 功能特色

- 🔥 **高强度加密**：采用 ChaCha20-Poly1305（AEAD），安全高效。
- 🧑‍💻 **灵活密钥**：口令经 SHA-512 哈希自动派生32字节密钥。
- 🔐 **文件级安全**：每个文件自动生成独立随机 nonce，杜绝重放。
- 🚀 **批量&多线程**：并发加速，适合大数量文件操作。
- ♻️ **原地覆盖**：直接覆盖原文件（操作前请备份重要数据！）
- 💻 **跨平台**：支持 Windows / Linux / macOS。

```bash
pip install pycryptodome
```

## ⏱️ 快速上手

1. **准备你的文件夹**  
   把要“加密”或“解密”的所有文件，放到同一目录或其子目录下。

2. **运行加密/解密脚本**

   ```bash
   python batch_chacha20_poly1305.py
   ```

3. **根据提示操作**

   - 输入操作类型：`e`（Encrypt加密）或 `d`（Decrypt解密）
   - 输入目标文件夹路径：比如 `./my_folder`
   - 输入口令（输入时不会显示在屏幕）

## 🛠️ 算法与原理

- 💡 **密钥派生**：口令 ➔ SHA-512 哈希 ➔ 截取32字节
- 🔑 **ChaCha20-Poly1305 加密**：每文件独立随机生成12字节 nonce
- 🗃️ **写入格式**：
  ```
  [nonce(12字节)][tag(16字节)][ciphertext]
  ```
- 🙌 **多线程处理**：显著提升批量加解密速度

---

## ⚠️ 注意与安全说明

- ❗**牢记口令**：遗失无法解密！
- 🌀 **原地覆盖**：本程序会覆写原文件，建议初用请做好数据备份！
- 🔒 **算法加密强度高**：ChaCha20-Poly1305为广泛认可的安全现代表达式。
- 🏠 **使用场景**：适合个人/企业本地归档或运维用途，不建议用于多人/云端多主密钥协作环境。
